Introduction

Visual Basic for Applications(VBA) allows Microsoft programs to communicate with each other based on events or actions that take place within those programs. This language is not only used in Office programs like Excel and PowerPoint, but also in programs like NotePad and Paint. I adapted it to create macros in Microsoft Excel.
